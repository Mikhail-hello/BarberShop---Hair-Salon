Данный проект представляет собой веб-платформу стартапа барбершопа, разработанную в рамках курсовой работы по дисциплине «Интернет-программирование». 
Сайт предназначен для презентации услуг барбершопа в онлайн-среде и формирования современного цифрового имиджа компании.

Основное назначение сайта — информирование потенциальных клиентов о барбершопе, его услугах, ценах, команде и актуальных трендах в бьюти-индустрии. 

Целью проекта является проектирование и разработка адаптивного и интерактивного веб-сайта барбершопа, который способствует привлечению новых клиентов, упрощает процесс ознакомления с услугами и их стоимостью
а также демонстрирует возможности современных веб-технологий в сфере услуг.

В ходе разработки проекта были использованы следующие технологии и инструменты:
HTML5 — для создания структуры веб-страниц;
CSS — для стилизации и оформления пользовательского интерфейса;
JavaScript — для реализации интерактивного функционала;
Bootstrap — для адаптивной вёрстки и создания кроссбраузерного дизайна;

Сайт выполнен в формате лендинг-страницы и ориентирован на удобство пользователя. Он корректно отображается как на персональных компьютерах, так и на мобильных устройствах.

Функциональные возможности сайта включают:
навигационное меню с адаптацией под мобильные устройства;
информационные блоки о стартапе, целях и целевой аудитории;
интерактивный калькулятор расчёта стоимости услуг;
форму обратной связи;

Проект может быть использован как основа для дальнейшего развития полноценной информационной системы барбершопа с возможностью онлайн-записи и расширенного функционала.
